# Digital Dice Project 🎲

## Overview

This is a **Digital Dice** project designed for the **Logic Design course**. It simulates a dice using a **7-segment display**, a **push button**, and an **Arduino Uno**. When the button is pressed, the dice rolls and displays a random number between 1 and 6.

## Components

* Arduino Uno
* 7-Segment Display (Common Cathode)
* Push Button
* Piezo Buzzer (for sound feedback)
* 9V Battery
* Resistors

## Features

* Generates a random dice number (1–6) on button press
* Visual display on a 7-segment display
* Audio feedback using a buzzer

## How It Works

1. The Arduino continuously checks the state of the push button.
2. When the button is pressed, a random number between 1 and 6 is generated.
3. The corresponding segments of the 7-segment display are activated to show the number.
4. The buzzer emits a short beep to indicate the roll.

## Code Usage

1. Open `DigitalDice.ino` in the Arduino IDE.
2. Connect the Arduino Uno to your computer.
3. Upload the code to the Arduino board.
4. Press the button to roll the dice.
